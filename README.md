# Employee Tracker

## Description
Do you need a new managing system to see what employees are in which department and who manages them? Look no further because with this application you can do it with ease! This app impliments a system where you can add new departments, add new employees, add the role of each employee and add the salary that they make. As well as querying through each department and employee. Simply run the code through the terminal and you'll have a selection of options of what you can do when setting up new employees, departments, or roles.
